# Play Console Release Notes 1.3.44

## de-DE

Verbesserte Berechnung und Darstellung der Bier-Kuehlzeit.

- Kuehlzeitberechnung auf ein wissenschaftliches freies-Konvektionsmodell umgestellt.
- Aktuelle Bier-Temperatur wird waehrend des laufenden Timers aus demselben Modell berechnet.
- Landscape-Layout verbessert, damit alle Temperaturregler sichtbar bleiben.
- Android-App-Bundle fuer Google Play aktualisiert.

## en-US

Improved beer cooling-time calculation and display.

- Cooling-time calculation now uses a scientific free-convection model.
- Current beer temperature during a running timer is calculated from the same model.
- Landscape layout improved so all temperature controls remain visible.
- Android App Bundle updated for Google Play.
